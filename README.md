# Better Console Line

A lightweight server-side Fabric mod that replaces Minecraft's basic console input with JLine.

It keeps command input readable while logs are being printed and works well in interactive terminals such as SSH and tmux.

## Features

* Supports command history and line editing
* Provides command completion
* Handles UTF-8 input reliably
* Restores the input line after log output
* Highlights warnings and errors with colors
